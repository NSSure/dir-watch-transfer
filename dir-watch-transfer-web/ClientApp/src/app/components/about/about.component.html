About
DirWatchTransfer allows for file and directory syncing, watching, and copying.

DirWatchTransfer allows for symbolic links to be created be a source and target directory. From those links watchers and syncs can be generated as you need. These can be configured to match your specific need/use case.

A detailed breakdown of each of the following features is provided below.

  • Symbolic links
  • Watchers
  • Directory sync
  • Logging
Watchers
Watch for file changes in real time based on user specified notification filters.

Watchers are fully customizable with configurable notification filters. When creating a watcher you can set what consitutes a file change for this specific watcher. Multiple watchers can be created for a single symbolic link however only ONE watcher at a time may be running. If you wish to switch watchers stop the currently running watcher and start your desired watcher.

The following fitlers can be applied to watchers.

  • File name
  • Directory name
  • Size
  • Last write
  • Last access
  • Creation time
  • Security

As of version 1.0.0 the following filters have not been testing and are not guarenteed to work property Directory name, Creation time, Security.

Sync
Schedule routine directory syncs to run as background processes.
